Vagrant-LEMP Stack
==================

Default LEMP development stack configuration for Vagrant.

Usage:
------

Download Vagrant from http://vagrantup.com.

Clone this repository

Go to the repository folder and launch the Vagrant box
'cd [repo]'  
'vagrant up'

What's Inside:
--------------

Installed software:
+ Ubuntu 12.04
+ nginx 
+ mysql
+ php
+ ajenti management console
+ git
